Question 1: What are the different ways to select an element in the DOM?

There are 5 ways in which you can select elements in a DOM using selectors.
1. getElementsByTagName();
2. getElementsByClassName();
3. getElementById();
4. querySelector();
5. querySelectorAll();

Question 2: What is the difference between innerHTML, innerText, and textContent?

১. innerHTML বর্ণনা: এই প্রপার্টি একটি উপাদানের মধ্যে থাকা HTML মার্কআপকে পায় বা সেট করে। এটি HTML ট্যাগ অন্তর্ভুক্ত করতে পারে, যার মানে আপনি এটি ব্যবহার করে একটি উপাদানে HTML কন্টেন্ট সন্নিবেশ করতে পারেন। ব্যবহার: পাওয়া: HTML কন্টেন্টকে একটি স্ট্রিং হিসেবে ফেরত দেয়। সেট করা: HTML কন্টেন্ট সেট করতে দেয়, যা ট্যাগ অন্তর্ভুক্ত করতে পারে।

Question 3: What is event delegation in the DOM?

ইভেন্ট ডেলিগেশন হল একটি শক্তিশালী কৌশল যা জাভাস্ক্রিপ্টে ইভেন্টগুলি আরও কার্যকরভাবে পরিচালনা করতে ব্যবহৃত হয়। এর মাধ্যমে, আপনি একটি প্যারেন্ট উপাদানে একটি ইভেন্ট লিসেনার যুক্ত করেন, যা তার শিশু উপাদানগুলির ইভেন্টগুলির জন্য কাজ করে।

Question 4: What is event bubbling in the DOM?

ইভেন্ট বুবলিং হল DOM-এর একটি মৌলিক ধারণা যা ইভেন্টগুলির প্রভাব কিভাবে উপাদানের হায়ারার্কিতে ছড়িয়ে পড়ে তা বর্ণনা করে। যখন একটি ইভেন্ট একটি নির্দিষ্ট উপাদানে ঘটে, তখন এটি প্রথমে সেই উপাদানে (লক্ষ্য) ট্রিগার হয় এবং তারপর এটি তার প্যারেন্ট উপাদানগুলির দিকে "বুবল" করে, প্যারেন্টগুলিতে সংযুক্ত যেকোনো ইভেন্ট লিসেনারকে ট্রিগার করে।

Question 5: How do you create, add, and remove elements using JavaScript?

জাভাস্ক্রিপ্ট ব্যবহার করে DOM-এ উপাদান তৈরি, যোগ এবং মুছে ফেলার প্রক্রিয়া নিচে সংক্ষেপে বর্ণনা করা হলো:
নতুন উপাদান তৈরি করতে document.createElement() মেথড ব্যবহার করা হয়।
তৈরি করা উপাদানটি DOM-এ যোগ করতে appendChild(), insertBefore(), বা append() মেথড ব্যবহার করা হয়।
DOM থেকে উপাদান মুছে ফেলতে remove() বা parentNode.removeChild() মেথড ব্যবহার করা হয়।
